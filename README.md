# Vibe — Платформа для поиска компании

**Vibe** — это веб-сервис для поиска компании на совместный досуг: походы, кино, настолки, путешествия и многое другое. Главная особенность платформы — безопасность и доверие, которые обеспечиваются через систему рейтингов и прозрачных отзывов.

---

## Основной функционал

Почти все ключевые фичи платформы уже реализованы и успешно работают:

- [x] **Современная авторизация:** Удобные всплывающие окна (модалки) для входа и регистрации, интегрированные с глобальным хэдером.
- [x] **Управление событиями:** Создание, просмотр, редактирование и безопасное удаление собственных мероприятий (с каскадным удалением зависимостей).
- [x] **Участие в активностях:** Запись на события, управление статусами заявок организатором.
- [x] **Профили пользователей:** Личные страницы с аватарами, биографией и статистикой.
- [x] **Система отзывов:** Возможность оставлять отзывы после встреч, формирование личного рейтинга пользователя (от 1 до 5 звёзд).
- [x] **Безопасность:** Проверка пароля при удалении аккаунта, JWT-токены для сессий, строгое разграничение прав (удалять/редактировать может только создатель).

---

## Стек технологий

Проект разделен на две независимые части (Бэкенд и Фронтенд), общающиеся по REST API.

**Backend:**
- **Python 3** + **FastAPI** — высокопроизводительный асинхронный фреймворк.
- **SQLite** + **SQLAlchemy** — база данных и ORM для взаимодействия с ней.
- **Pydantic** — валидация данных.
- **Pytest** — написание и запуск автоматических тестов.
- **Passlib & JWT** — хэширование паролей и генерация токенов доступа.

**Frontend:**
- **React.js (JavaScript)** — библиотека для построения современных UI.
- **Axios** — HTTP-клиент для запросов к бэкенду.
- **React Router** — клиентская маршрутизация.
- **Vanilla CSS** — кастомная дизайн-система с использованием CSS-переменных, Glassmorphism и современных анимаций.

**DevOps & Инструменты:**
- Git, GitHub Actions, Docker.

## Запуск проекта локально (Local Setup)

Для того чтобы развернуть проект на своём компьютере, вам потребуется Python (3.9+) и Node.js (16+).

### 1. Запуск Backend-сервера
Перейдите в корневую папку проекта и выполните следующие шаги:
```bash
# 1. Установка зависимостей
pip install -r requirements.txt

# 2. Запуск сервера FastAPI (по умолчанию порт 8000)
python -m uvicorn backend.app.routes.routes:app --reload --port 8000
```
После запуска, интерактивная документация по API (Swagger) будет доступна по адресу: `http://localhost:8000/docs`.

### 2. Запуск Frontend-сервера
Откройте новое окно терминала, перейдите в папку `frontend` и запустите React:
```bash
# 1. Переход в папку фронтенда
cd frontend

# 2. Установка npm-зависимостей
npm install

# 3. Запуск локального dev-сервера
npm start
```
Приложение автоматически откроется в браузере по адресу: `http://localhost:3000`.

## Тестирование
В проекте настроены юнит-тесты для бэкенда. Чтобы прогнать тесты, находясь в корневой папке проекта, выполните:
```bash
python -m pytest backend/tests/ -v
```
